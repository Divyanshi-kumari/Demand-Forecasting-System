# Demand Forecasting System

A machine learning based web application that predicts future demand and displays results through an interactive dashboard.

## Features
- Demand forecasting
- Interactive graphs
- Dashboard visualization
- Data-based prediction
- User-friendly interface

## Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000
